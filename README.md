# Analytical-Satsky
Modelling satellite constellations seen from astronomical observatories

### Installation

Simply git clone and pip install locally

```
git clone https://github.com/nicolas-cerardi/Analytical-Satsky.git
cd analytical-satsky 
pip install -e .

```

### Running

See the demo notebooks:
- [notebooks/compute_satellite_occupancy.ipynb](https://github.com/nicolas-cerardi/Analytical-Satsky/blob/main/notebooks/compute_satellite_occupancy.ipynb)
- [notebooks/satellite_density_maps.ipynb](https://github.com/nicolas-cerardi/Analytical-Satsky/blob/main/notebooks/satellite_density_maps.ipynb)
